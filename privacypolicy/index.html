Denna applikation läser av telefonens state på så vis att den kollar vilken operatör som används för att kunna ta fram rätt MMI-kod för att sköta vidarekopplingen korrekt. Den data som lagras i appen är:

  • Telefon nummer du anger
  • Om du för tillfället vidarekopplar eller ej
  • Vilken senaste tid du använde var
Ingen information lagras utanför appen. Det vill säga, tar du bort appen så försvinner även all info. Ingen tredje part kommer åt din information, om du inte aktivt tillåter den.
